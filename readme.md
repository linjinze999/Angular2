# Angular2 学习文档
#### 导语
>  Angular2是一套全新的Angular框架。之前的Angular1由于底层实现的问题，在性能上有所瑕疵，为了解决这个问题，Google推翻Angular1重新写了Angular2，因此1和2是完全不同的。Angular2文档很早之前就有，但正式代码去年才发布，因此组件库也比较少，目前我只列出了一些学习资料以供学习。
